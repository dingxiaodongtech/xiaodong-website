## MCP 工具速查（bigdata-mcp-server）

> 所有工具均通过 **`user-bigdata-mcp-server`** 调用，需先配置 RefreshToken。

### ✅ 已替换工具

| 旧工具 | 新工具（bigdata-mcp-server） | 用途 | 适用路径 |
|-------|---------------------------|-----|---------|
| `query_table_by_semantic` | **CLI**：`python3 .cursor/skills/bigdata-conan-text2sql/scripts/token_mcp_stdio.py search <keyword>`<br>退化方案：`metadata_searchTables(keywords=..., dbTypes=["Hive"])` | 语义搜索定位目标表 | 所有路径（找表） |
| `query_table_full_lineage` | **两步**：① `metadata_getTableDetail(table=..., tableType="Hive")` 取 `tableId`<br>② `metadata_getTableLineage(tableId=<int>, upperDepth=5, lowerDepth=2)` | 获取完整血缘链路 | 路径 A-3 |
| `get_change_sql_content` | `change_getTaskScriptContent(taskName="xxx.hql")` ⚠️ **参数是任务名（含后缀），不是表名**；任务名可从 `metadata_getTableDetail` 返回的 ETL 任务字段中获取 | 获取表的 ETL 脚本 | 路径 A-4 |
| `mario_execute_trino_query` | `adhoc_submitQuery(engine="Trino", catalog="hive_f04", database=<首个库名>, sql=...)` → 返回 `queryId` | 提交 Trino SQL | 所有路径 |
| `mario_get_sql_query_result` | `adhoc_getQueryResult(queryId=<int>, timeoutSeconds=30)` | 获取查询结果（轮询） | 所有路径 |
| `mario_execute_doris_query` | `adhoc_submitQuery(engine="Doris", database=<库名>, sql=...)` → `adhoc_getQueryResult` | 执行 Doris SQL（广告数据） | 路径 B（投放归因） |

### ❌ 暂无替代工具

| 旧工具 | 说明 | 临时处理方式 |
|-------|------|------------|
| `get_job_change_history` | bigdata-mcp-server 无作业历史变更记录接口 | 路径 A-5 降级为手动在 Change 平台查询变更日志，或通过 `change_getTaskDetail` 看任务基本信息 |
| `query_table_column_relationships` | bigdata-mcp-server 无字段级关联工具 | 路径 A-6 降级为查 ETL 脚本（`change_getTaskScriptContent`）后人工分析字段映射关系 |

## 常见异常类型处理指引

| 异常类型 | 优先检查项 | 归因路径 |
|---------|-----------|---------|
| 数据量偏低（ETL） | ODS 分区 → DWD 过滤 → JOIN 类型 | 路径 A |
| 转化率持续下降 | 渠道结构 + 用户城市构成 + 漏斗各环节 | B + C + D |
| 续报率下降 | 结课用户组成 + 完课率 + 老师服务 | C + D |
| 完课率低 | 用户地域 + 开课时间 + 老师分配 + 完课率区间分层 | C + G |
| SLOGAN 外呼名单 | 高潜筛选 + 流失预警 + 召回候选 | 路径 E |
| 新渠道效果差 | 渠道用户质量 + 城市构成 + 完课行为 | B + C + D |
| 这批用户的画像是什么 | 课程状态 + 学习行为 + 地域 + 续报意向多维分层 | 路径 G |
| 人群质量评估 | 系统课在课率 + 完课率分布 + 付费时长 + 扩科潜力 | 路径 G |
| 精细化人群分层运营 | 按续报紧急度 + 学习活跃度 + 科目组合细分 | G + E |
