# 研究发现和技术决策

## 技术栈选择
- **框架**：Next.js (App Router) - 全栈开发
- **样式**：Tailwind CSS - 快速开发响应式UI
- **数据库**：Supabase (PostgreSQL)
- **AI SDK**：ai + @ai-sdk/react（需安装）
- **认证**：自建用户系统（邮箱注册/登录）

## 2026-04-22 需求升级

### 原需求
- 一站式商业平台官网（Landing Page）

### 新需求（确认）
- **产品**：AI 自适应数学教育平台
- **学科**：数学（小学一年级 ~ 初三）
- **教学形式**：AI 互动教学 + 智能诊断（错因驱动 + 分步讲解）
- **付费模式**：3 次免费体验，之后付费解锁
- **用户规模**：初期不确定，需可扩展

### 数据库表设计
- `users` - 用户信息（姓名、年级、邮箱、密码hash、会员状态、创建时间）
- `courses` - 课程（年级、名称、描述、封面图）
- `chapters` - 章节（所属课程、序号、名称、知识点）
- `questions` - 题目库（所属章节、题目内容、正确答案、难度、知识点标签）
- `learning_records` - 学习记录（用户ID、题目ID、用户答案、是否正确、AI诊断结果、用时）
- `free_usage` - 免费体验计数（用户ID、已用次数、上限3次）

### 用户上传文件分析
- `AI智能诊断自学demo课.html` - 完整的AI自适应教学Demo
  - 左右双面板布局：左侧AI监控台 + 右侧学生课堂
  - 包含7道数学题（排水法、表面积、相遇问题等）
  - 错因驱动诊断 + 分步讲解梯子
  - 微信风格对话交互
  - 可视化动画（水箱、长方体3D）

### 页面路由规划
- `/` - 首页（教育平台介绍 + 体验入口）
- `/login` - 登录
- `/register` - 注册
- `/courses` - 课程列表（按年级分）
- `/courses/[id]` - 课程详情（章节列表）
- `/learn/[questionId]` - AI 互动学习页面
- `/dashboard` - 个人中心（学习进度、错题本）
- `/pricing` - 付费页面
