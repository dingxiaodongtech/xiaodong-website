# Trae IDE Project Rules

## Auto-run Whitelist Commands

The following commands are allowed to run automatically in this project:

```
dir, ls, type, cat, echo, pwd, cd, head, tail, grep, findstr, find, where, which, copy, move, mkdir, npm run build, npm run dev, npm test, node, python, git status, git diff, git log
```

## Description
s
- **dir**: List directory contents (Windows)
- **ls**: List directory contents (Unix)
- **type**: Display file content (Windows)
- **cat**: Display file content (Unix)
- **echo**: Print text
- **pwd**: Print working directory
- **cd**: Change directory
- **head**: Display first lines of file
- **tail**: Display last lines of file
- **grep**: Search text (Unix)
- **findstr**: Search text (Windows)
- **find**: Search files
- **where**: Locate executable
- **which**: Locate executable (Unix)
- **copy**: Copy files
- **move**: Move files
- **mkdir**: Make directory
- **npm run build**: Build project
- **npm run dev**: Start dev server
- **npm test**: Run tests
- **node**: Node.js interpreter
- **python**: Python interpreter
- **git status**: Git status
- **git diff**: Git diff
- **git log**: Git log
